Это приложение, созданное на React, осуществляет поиск по github.com по вводимому имени пользователя, выводит список подходящих аккаунтов и позволяет открыть каждый из них. К каждому аккаунту подтягивается аватар, юзернейм и прочие параметры, которые присутствуют на странице выбранного пользователя.

При написании использовался исключительно классовый подход с помощью хуков useState, useContext, useReducer, useEffect. Так же, использовано axios, react-router-dom, bootstrap.

Приложение создано за результатами прохождения курса: https://www.udemy.com/course/react-2020-complete-guide/
